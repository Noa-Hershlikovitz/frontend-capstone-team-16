# frontend-capstone-team-16
This project is built by Team 16


## Description
Build Me is a website that anyone can use to create their own CV.  
In the website you can see samples of CV's that you can create in the website and which companies their workers used our website.  
You can read about our company and get some FREE tips from us.  
You can see Reviews from our customers and leave your review after using our website  
The main attraction is "CV-Creator" page when you can really create your own CV and see in real-time how your CV will look and in the end you can print/save the CV you created for future use.


## Features
* CV Creator that update in real-time
* Dark/Light mode
* Leave you own Review after CV creation
* Free tips for CV creation
